# TODO API
This is a simple TODO API, where users can create, read, update and delete todos.

- This application was built using NodeJs and MySQL

## How to Setup
1. After cloning the repo on the terminal, install the dependencies using:
```
npm i
```
2. Start the Server
```
node app.js
```
